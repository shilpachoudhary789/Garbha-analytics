# Garbha Analytics – Biomedical Data Analysis

This project focuses on analyzing biomedical datasets to understand patterns, distributions, and statistical insights related to maternal and offspring data.

## Tools Used
Python (Pandas, NumPy, Matplotlib)
SQL
Power BI
Tableau

## Dataset
The project includes biomedical datasets such as:
- Mother dataset
- Offspring dataset
- Elisa dataset
- Reports and dashboards datasets

## Analysis Performed
• Data cleaning and preprocessing  
• Exploratory Data Analysis (EDA)  
• Statistical analysis including mean, median, variance, and standard deviation  
• SQL queries for data extraction and analysis  
• Data visualization using Power BI and Tableau dashboards  

## Key Insights
• Identified patterns in biomedical variables  
• Analyzed distribution of important biomarkers  
• Built dashboards to visualize biomedical data trends  
• Derived insights to support data-driven understanding of the dataset
